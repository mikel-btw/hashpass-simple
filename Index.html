¿Cómo funciona?

01

Combinas dos datos

Introduces tu contraseña maestra y el nombre del sitio. Esos dos valores son todo lo que se necesita. Nunca se almacenan en ningún lado.

02

PBKDF2-SHA512 hace el trabajo

Tu navegador ejecuta 100.000 iteraciones del algoritmo de derivación de claves PBKDF2 con hash SHA-512. Es matemáticamente costoso de revertir o de atacar por fuerza bruta.

03

Resultado determinístico

Los mismos dos datos de entrada producen siempre los mismos bytes de salida, en cualquier dispositivo y momento. La contraseña se codifica con el conjunto de caracteres que elijas.

Garantías técnicas

Privacidad y
Seguridad

Sin almacenamiento

Ninguna contraseña se guarda en disco, cookies, localStorage ni ningún otro mecanismo de persistencia.

Solo local

Todo el cómputo ocurre dentro de tu navegador usando la Web Crypto API nativa. No existe backend.

Sin red

Ningún dato tuyo se transmite a ningún servidor, API externa ni tercero de ningún tipo.

Código auditable

El código fuente es completamente legible desde el navegador. No hay minificación ofuscada ni lógica oculta.
